# Exemplo de Contrato de Sprint

Objetivo do sprint:

- Adicionar citações visíveis aos resultados de perguntas e respostas baseados em fontes (*grounded Q&A*)

Considera-se concluído quando:

- O usuário faz uma pergunta
- A aplicação retorna uma resposta
- Pelo menos uma citação é exibida
- Ao clicar em uma citação, a localização correspondente da fonte é aberta na visualização do documento
