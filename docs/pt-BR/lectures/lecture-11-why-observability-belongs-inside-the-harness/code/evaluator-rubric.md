# Exemplo de Rubrica de Avaliação

Utilize uma pontuação de 1 a 5 para cada dimensão:

- **Fundamentação (*Grounding*)**: as respostas estão claramente vinculadas às fontes importadas?
- **Qualidade das citações**: as referências às fontes são visíveis e específicas?
- **Funcionalidade**: o usuário consegue concluir o fluxo completo de pergunta e resposta?
- **Coerência do produto**: o fluxo de trabalho parece integrado e consistente?
