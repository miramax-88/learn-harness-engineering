# Rúbrica do Avaliador (Evaluator Rubric)

Use esta rúbrica após a implementação e antes da aceitação final.

| Categoria | Pergunta | Pontuação (0-2) | Notas |
| --- | --- | --- | --- |
| Correção | O comportamento implementado corresponde ao recurso solicitado? |  |  |
| Verificação | As verificações exigidas foram realmente executadas, com evidências? |  |  |
| Disciplina de Escopo | A sessão permaneceu dentro do escopo do recurso escolhido? |  |  |
| Confiabilidade | O resultado sobrevive à reinicialização ou reexecução sem reparos? |  |  |
| Manutenibilidade | O código e a documentação estão claros o suficiente para a próxima sessão? |  |  |
| Prontidão para Entrega | Uma nova sessão pode continuar o trabalho apenas a partir dos artefatos do repositório? |  |  |

## Veredito

- Aceitar
- Revisar
- Bloquear
