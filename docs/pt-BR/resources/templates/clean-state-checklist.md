# Checklist de Estado Limpo (Clean State Checklist)

- [ ] O caminho de inicialização padrão ainda funciona.
- [ ] O caminho de verificação padrão ainda roda.
- [ ] O progresso atual está registrado no log de progresso.
- [ ] O estado do recurso reflete o que está realmente passando versus o que não foi verificado.
- [ ] Nenhuma etapa inacabada foi deixada sem documentação.
- [ ] A próxima sessão pode continuar sem a necessidade de reparos manuais.
